Base De Datos 2023

Modelo entidad relación

Conceptos básicos

Dato:Palabras y números que por si solo no tienen significado.
Base de datos: Es una colección de datos relacionadas, ordenadas y organizadas para que tengan sentido sus datos contenidos y puedan ser consultados.
Entidad:
Es un objeto que representa algo del mundo real, que puede ser descripto en una base de datos.
Esta entidad puede representar un objeto con una existencia conceptual, o con una existencia física, por ejemplo, un auto, un puesto de trabajo, una materia escolar, una persona, etc.
Así también una entidad físisca es denominada entidad concreta, a una conceptual la llamaremos entidad abstracta y éstas se representarán en un modelo de entidad relación con un rectángulo.
Atributos: Un atributo es una propiedad o una característica que describe a una entidad, color, nombre, dirección etc.
El atributo se representa en el modelo de entidad relación mediante un óvalo. Una entidad puede tener varios atributos y un atributo puede tener varios registros.
